¿Qué es la Web?

La Web es un sistema de páginas interconectadas que funcionan a través de Internet y permiten compartir información en todo el mundo.

Conceptos básicos

  • HTML estructura el contenido
  • CSS da estilo a la página
  • JavaScript agrega interactividad
